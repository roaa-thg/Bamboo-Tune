# Contributing

Thanks for your interest in BAMBOO TUNE!

## How to contribute
1. Fork the repository and create a branch: `git checkout -b feature/your-feature`.
2. Make your changes. Keep notebook outputs cleared before committing
   (`jupyter nbconvert --clear-output --inplace src/bamboo_tune_pipeline.ipynb`).
3. Never commit secrets, model weights, or the full private dataset.
4. Open a pull request describing what you changed and why.

## Reporting issues
Use GitHub Issues for bugs and feature requests. For security issues, see
[SECURITY.md](SECURITY.md) instead.
